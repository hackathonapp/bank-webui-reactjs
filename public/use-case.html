Branch & ATM Use Case Solution About

Build an app for Bank to open account online in easy steps which includes capture primary client data, qualify applicant, validate KYC.

User will enter the basic details like Name, email, phone, address, profession etc. In the next step, user will upload the KYC documents (identification and address) which helps in validating user info.

It will be good if with cognitive (OCR) service the documents can be validated, and OTP verification is done to confirm phone/email.

Further enhancements that may be done - Photo Signature and captcha on T&C page, PIN/Password stored in encrypted format. Acknowledgement number to be provided or save details in case application abandoned.

Hosted on Red Hat Openshift
